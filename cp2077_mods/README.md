# Grand Theft Cyberpunk - Abyssal Assets

## Mod List
- Metaconscious Singularity Node (MSN)
- Sephirotic Court Framework
- Hell Campaign & Space Quests
- MSN Magic & Jedi Systems
- Abyssal Assets & Local Economy

## System Requirements
- Cyberpunk 2077 v2.1+
- RED4ext & redscript installed
- REDmod enabled
- Local AI Inference: Ollama with 8GB+ VRAM (for FP16 inference on primary nodes, INT8/INT4 on secondary)
- Minimum 16GB System RAM (32GB Recommended)

## Installation Instructions
1. Ensure Cyberpunk 2077 and all script extenders (RED4ext, redscript, TweakXL, ArchiveXL) are up to date.
2. Extract the archive into your Cyberpunk 2077 directory.
3. Start the local inference server (`msn-cyberpunk`).
4. Launch the game via REDmod.
